Project Overview
PhilanthroLink is an Android application developed to connect Non-Governmental Organizations (NGOs) and Volunteers through a secure, transparent, and user-friendly platform. NGOs can create donation requests while volunteers can discover organizations, contribute donations, verify NGOs using QR codes, provide reviews, and track donation activities. The application leverages Firebase services to provide secure authentication, real-time database synchronization, and a modern Android user experience built with Kotlin and Material Design.
