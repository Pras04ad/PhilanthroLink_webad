# Project Overview

PhilanthroLink is an Android application designed to connect **Non-Governmental Organizations (NGOs)** and **Volunteers** through a secure, transparent, and user-friendly platform. The application enables NGOs to create and manage donation requests while allowing volunteers to discover organizations, contribute resources, verify NGOs using QR codes, and participate in a trusted community through reviews and feedback.

The application is built using modern Android development technologies, including **Kotlin**, **Firebase Authentication**, **Firebase Realtime Database**, and **Material Design**. These technologies provide secure user authentication, real-time synchronization, and an intuitive user experience.

The purpose of the Web Advertisement is to promote the application, introduce its major features, showcase the user interface through screenshots, and provide users with a publicly accessible landing page where they can learn more about the application.
