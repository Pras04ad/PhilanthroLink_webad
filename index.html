About PhilanthroLink

PhilanthroLink is an Android application designed to bridge the gap between Non-Governmental Organizations (NGOs) and Volunteers. Organizations can publish donation requests while volunteers discover NGOs, view live requirements, donate resources, scan QR codes, and participate through a transparent review system. Built using Kotlin, Firebase Authentication and Firebase Realtime Database, the application provides a modern platform for efficient donation management.
