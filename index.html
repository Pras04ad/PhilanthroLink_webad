About PhilanthroLink

A modern Android application designed to simplify charitable giving and volunteer engagement through a secure cloud-based platform.

Mission

Provide a transparent digital platform where NGOs can request help and Volunteers can easily contribute resources.

Vision

To build stronger communities by making charitable giving simple, accessible, and transparent.

Technology

Built using Kotlin, Firebase Authentication, Firebase Realtime Database, and Material Design.

Key Features
